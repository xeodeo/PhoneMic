# PhoneMic v1.2

## Mejoras y correcciones

### Corrección de bug: reconexión USB al día siguiente
Al volver a conectar el teléfono después de haberlo desconectado, la app ya no pide revocar y re-autorizar la depuración USB. Ahora detecta cuando el dispositivo está en estado `offline` y reinicia el daemon ADB automáticamente para restablecer la conexión sin perder la autorización existente.

### Notificación de actualizaciones
Tanto la app de Windows como la app de Android avisan cuando hay una nueva versión disponible en GitHub. La notificación aparece al iniciar y se puede cerrar sin necesidad de actualizar.

### Nuevo menú de desarrollo (`dev.bat`)
Script con menú interactivo para facilitar el desarrollo:
- Instalar APK directamente al teléfono por USB
- Compilar todo (APK + EXE + Instalador)
- Compilar solo APK o solo EXE
- Instalar dependencias Python con un solo clic

### Detección automática de ADB mejorada
Los scripts de Windows ahora buscan `adb.exe` en múltiples ubicaciones automáticamente: junto al script, WOMic, Android SDK. Ya no falla si no tienes WOMic instalado.

### Instalador inteligente
El instalador detecta si PhoneMic ya está instalado y muestra el diálogo correspondiente:
- **Misma versión:** pregunta si deseas reinstalar
- **Versión anterior instalada:** ofrece actualizar
- **Versión más nueva instalada:** advierte antes de hacer downgrade

### Corrección del empaquetado del EXE
Corregido el comando de PyInstaller para incluir correctamente todos los archivos de PySide6 (`--collect-all PySide6`). El EXE instalado ya no falla al iniciar por módulos de Qt faltantes.
