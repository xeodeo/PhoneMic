# PhoneMic v1.3

## Mejoras y correcciones

### Compatibilidad con Python 3.14
Corregido un crash de PyInstaller al compilar el EXE en Python 3.14. El problema era que `--collect-all PySide6` intentaba importar `PySide6.support` en un subproceso aislado que fallaba con la versión más reciente de Python. Ahora se usan los hooks internos de PyInstaller para PySide6, que funcionan correctamente en todas las versiones.

### EXE más liviano (~50% menos)
El instalador ahora usa `PySide6-Essentials` en vez del paquete completo de `PySide6`. El paquete completo incluye módulos que la app no usa (QtWebEngine, QtMultimedia, Qt3D, QtBluetooth, etc.). Con solo los módulos esenciales el EXE generado pesa aproximadamente la mitad.

### Dependencias completas en requirements.txt
El archivo `windows/requirements.txt` ahora incluye todas las dependencias necesarias para compilar y ejecutar la app, incluyendo `pystray` que faltaba anteriormente.

### Opción de instalación de dependencias en dev.bat
El menú de desarrollo (`dev.bat`) ahora incluye una opción para instalar todas las dependencias de Python automáticamente con un solo clic, incluyendo la migración de `PySide6` a `PySide6-Essentials`.
